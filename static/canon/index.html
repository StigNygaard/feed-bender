Canon related news feeds

Some custom Canon photography related news feeds. All available in both RSS and JSON Feed format.

Canon Rumors - Essential posts only

Especially if you follow Canon Rumors via their own official RSS news feed (or their newsletter), it has lately become a very "spammy" site, with lots of irrelevant and low-quality posts to - I assume - attract "random" searchers and get more "affiliate clicks". But the interesting posts are still there, I just don't want my news-reader to be flooded with all the other noise. So I have tried creating a slimmed-down custom news-feed with only the "interesting" posts.

"My feed" is based on the official news feed, but I'm filtering it trying to keep only the essential posts. Currently all posts in some categories are skipped - and for another group of categories I'm also skipping some posts, but only if no canon relevance is detected in any of the (usually multiple) categories/tags for the posts. Canon Rumors use of categories/tags feels inconsistent and/or difficult and complex, so I have done many adjustments to the filtering so far. I will probably still do adjustments, but generally I think my feed works pretty well now.

Canon Rumors Forum - Latest new threads

Unlike the official forum feed which contains every new comment posted in Canon Rumors Forum, this custom-feed tries keeping track of only the latest new threads (topics) created in the forum. This feed also tries to avoid the threads that are created to be "comment-sections" for news-posts on the main-site. While "comment-section"-threads for Craig's news-posts usually have a pattern easy to identify, it is harder with the "comment-section"-threads for Richard's posts, so I am now ignoring all threads started by "Richard CR" in the forum. A bit unfair to Richard, but I think it is pretty rare he starts other topics in the forum.
